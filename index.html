Welcome to Web Design!

This one semester course at Park East High School aims to prepare students in basic web development skills.
Topics covered:

  • Introduction to computers and the internet
  • Ethical Problems in the CS world
  • Writing a CS tailored cover letter
  • Basics in HTML
  • Basics in CSS
  • Advanced techniques in HTML and CSS using Bootstrap
  • Basics in Javascript
  • Creating webpages from scratch using the above languages
